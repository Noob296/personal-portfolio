# Javed Khan | Personal Portfolio Website

This is a simple, static personal portfolio website for Javed Khan, showcasing his skills, projects, and contact information. It also includes a fun, interactive Tic-Tac-Toe game.

## Features

- **About Me**: A brief introduction to Javed Khan.
- **Skills**: Highlights key technical skills in a grid format.
- **Projects**: Showcases a few notable projects with short descriptions.
- **Interactive Game**: Includes a playable Tic-Tac-Toe game for engagement.
- **Contact Information**: Provides email and GitHub profile links.
- **Responsive Design**: Basic responsiveness for viewing on different devices.

## Technologies Used

-   **HTML5**: For structuring the web content.
-   **CSS3**: For styling and layout.
-   **JavaScript**: For interactive elements, including the Tic-Tac-Toe game.
-   **Font Awesome**: For skill icons.

## How to Run Locally

This is a static website, so you don't need any special server setup. You can open it directly in your web browser.

1.  **Clone the Repository** (or download the zip file):
    ```bash
    git clone <repository_url>
    cd PortfolioProject
    ```
2.  **Open `index.html`**: Navigate to the `PortfolioProject` directory and open the `index.html` file with your preferred web browser.
    ```bash
    # Example for Linux/macOS
    open index.html
    # Example for Windows
    start index.html
    ```

## Project Structure

```
PortfolioProject/
├── index.html      # Main HTML file for the website
├── style.css       # CSS file for styling
└── script.js       # JavaScript file for interactive elements (Tic-Tac-Toe)
```

## Customization

-   **Content**: Edit `index.html` to update your personal information, skills, projects, and contact details.
-   **Styling**: Modify `style.css` to change colors, fonts, and layout.
-   **Tic-Tac-Toe**: The game logic is in `script.js`.
